# interesting-algorithms

Interesting algorithms for discussion, written in Python.

Each notebook is a self-contained, teaching-oriented walkthrough with a pure-Python implementation, step-by-step examples, and references to the original papers.

## Algorithms

| Algorithm | Description | Notebook |
|-----------|-------------|----------|
| Sequitur | Infer a context-free grammar from a sequence in linear time | [notebooks/sequitur.ipynb](notebooks/sequitur.ipynb) |
| Arithmetic Coding | Encode an entire message as a single fractional number; elegant and optimal | [notebooks/arithmetic_coding.ipynb](notebooks/arithmetic_coding.ipynb) |
| Reservoir Sampling | Uniformly sample k items from a stream of unknown length in one pass | [notebooks/reservoir_sampling.ipynb](notebooks/reservoir_sampling.ipynb) |
| Dancing Links (DLX) | Knuth's technique for solving exact cover problems (Sudoku, pentominoes) | [notebooks/dancing_links.ipynb](notebooks/dancing_links.ipynb) |
